# Changelog

## 2.0.1+3

- Atualização das chamadas de seleção de ficheiros para a API estática do `file_picker` 11.
- Adição do parâmetro visual `eyebrow` ao componente `SectionHeader`.
- Substituição de um ícone indisponível por um ícone estável do Material.
- Substituição do teste-modelo gerado pelo Flutter, que referia `MyApp`, por um teste real do EnglishForge.
- Correção dos avisos de estilo apontados no reprodutor e na biblioteca de materiais.

## 2.0.0+2

- Reconstrução em arquitetura modular.
- Correção do conflito `timezone`: dependência direta atualizada para `0.11.1`.
- Currículo completo A1–C2 em JSON.
- SQLite com 12 tabelas e backup JSON.
- Calendário, modo foco, alarmes, faltas e dívida de estudo.
- Speaking Lab com gravações e comparação histórica.
- Vocabulário com revisão espaçada.
- Editor e executor de quizzes com feedback.
- Banco de exercícios com respostas-modelo.
- Biblioteca antiacúmulo.
- Immersion Player com velocidade, loop A–B e marcadores.
- Painel de progresso responsivo.
- Workflow GitHub Actions com Flutter 3.44.4, API 36, AGP 8.11.1 e Gradle 8.13.
- Testes unitários para repetição espaçada e cálculo de sequência.
